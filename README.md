## Installation
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/dhrm9/Rise.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Rise
   ```

3. Install dependencies:
   ```bash
   npm i 
   ```

4. Start the project:
   ```bash
   npm start
   ```
